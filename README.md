# MongoDB Student Database Project

This project demonstrates how to connect Python with MongoDB Atlas and retrieve student data.

## Technologies Used
- Python
- MongoDB Atlas
- PyMongo

## Database Structure
Database: student_db  
Collection: students

## Sample Data
Name: Ravi  
Marks: 85  
Branch: CSE  

Name: Priya  
Marks: 70  
Branch: ECE  

## Description
The Python script connects to MongoDB Atlas using PyMongo and retrieves student records from the database.
